# UI Layer Documentation

The UI layer includes user-facing interface for course enrollment and analytics. The views are provided for students, teachers and administrative workers.

## UI Presenter

The **UIPresenter** module provides the web-based front-end for enrollment functionality.
It offers screens and interaction flows for students to enroll in or drop courses and enables teachers to review and manage student enrollment lists for their courses.
Notifications about the result of operations are also displayed.

![](embed:UIPresenter)

## Statistics UI

The **StatisticsUI** module provides user interfaces for querying enrollment data and visualizes analytics on request from administrative workers.
The UI enables the staff to specify data query parameters according to their requirements.
The UI displays results from statistical analyses and generates interactive graphs.
