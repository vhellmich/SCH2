## Report HTML

**Purpose**  
Delivers the reporting dashboard for managers who monitor enrolment and capacity trends.

**Main responsibilities**
- Present aggregated metrics, visualisations, and recommendations in a clear layout.
- Allow filtering, drill-down, and export initiation from within the interface.
- Communicate status messages and guidance from the reporting workflows.

**Collaborations**
- Requests datasets, charts, and export artefacts from `Reporting Business`.
